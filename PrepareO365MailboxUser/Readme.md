# Remove Unverified Email Domain from users

This PowerShell script removes certain unverified email domain from Active directory Mailbox users account.
So the mailbox can be migrated to office365.

Feel free and read more about it here
https://www.powershellcenter.com/2020/11/01/you-cant-use-the-domain-because-its-not-an-accepted-domain-for-your-organization/
